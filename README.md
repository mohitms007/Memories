# Memories
The App is called "Memories" and it is a simple photo media app that allows users to post interesting events that happened in their lives. (Based on MERN stack)

### Technologies used -: 
  1. Node JS
  2. React
  3. MongoDB Atlas
  4. Heroku
  5. Netify
  6. React REDUX
